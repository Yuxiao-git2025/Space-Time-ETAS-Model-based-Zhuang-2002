# Space-Time-ETAS-Model-based-Zhuang-2002
This is a MATLAB reproduction of the `spatial-temporal ETAS model` proposed by Teacher Zhuang in 2002, which is more concise and easier to understand. <br>
---
* It supports `custom` input and output. The code uses `global` variables, which might be one of the reasons for the slow running speed. Therefore, please use it with caution.<br>
* The `APP` program has been designed. Please contact the author if needed.<br>
* Reference: <br>
Zhuang, J., Ogata, Y., & Vere-Jones, D. (2002). Stochastic Declustering of Space-Time Earthquake Occurrences. Journal of the American Statistical Association, 97, 369–380. https://doi.org/10.1198/016214502760046925
